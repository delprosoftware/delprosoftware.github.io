# Technican Job List

## Job List

This list displays all current jobs assigned to you, showcasing all relevant details for each entry.

 - Call Status
 - Job Number
 - Job Type
 - Client Call Number
 - Allocation
 - Additional Notes
 - Attend By SLA
 - Close By SLA
